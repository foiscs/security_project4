🎯 취약점 정보

  • CVE: CVE-2022-22965 (Spring4Shell)
  • 영향받는 버전: Spring Framework 5.1.2
  • 공격 벡터: @ModelAttribute를 통한 ClassLoader 접근
  • 위험도: Critical (CVSS 9.8)
  • 취약 조건: JDK 9+, Tomcat WAR 배포, @ModelAttribute 사용

🔧 API 엔드포인트

  • GET /api/v1/vehicles - 차량 목록 조회
  • GET /api/v1/vehicles/{id} - 차량 상세 조회
  • POST /api/v1/rentals - 렌탈 시작
  • PATCH /api/v1/rentals/{id} - 렌탈 반납
  • POST /api/v1/reservations - 예약 생성
  • POST /api/v1/payments - 결제 처리
  • POST /api/v1/users - 사용자 등록
  • GET /api/v1/vehicles/{id}/door - 도어 상태 조회

⚠️ 교육 및 보안 연구 목적으로만 사용하세요 ⚠️
